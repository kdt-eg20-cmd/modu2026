# 1년 차 획득 가능 시장(SOM) 정의 보고서

## 건강보조식품 성분·가격 비교 플랫폼

---

## Executive Summary

| 구분 | 규모 | 근거 |
|---|---|---|
| **TAM** (한국 건기식 정보 시장 전체) | **500~1,500억 원** | 글로벌 TAM 10~25억 USD의 한국 비중 |
| **SAM** (커머스 연동 모델, 부가 수익 포함) | **5~11억 원/년** | 기본 시나리오, 인지도 확보 전제 |
| **SOM** (1년 차 획득 가능 시장) | **0.5~1.0억 원/년** | 본 보고서 추정 |

1년 차 SOM은 SAM의 약 **10~15%** 수준이다. 이는 신규 플랫폼이 브랜드 인지도 0에서 출발하여, MVP 출시 후 실질적으로 9~10개월 간 운영하며 획득할 수 있는 현실적 규모를 의미한다.

> **핵심 판단:** 1년 차에 독립 사업으로서 수익성을 확보하기는 어렵다. 그러나 **사용자 기반 확보, 데이터 축적, PMF(Product-Market Fit) 검증**이라는 전략적 자산을 1년 차에 구축할 수 있다면, 2~3년 차의 수익 전환을 위한 충분한 기반이 된다.

## 1. SOM 추정 프레임: SAM에서 어떻게 축소되는가

### 1.1 SOM 축소 논리

SAM(유효 시장)에서 SOM(획득 가능 시장)으로의 축소는 다음 세 가지 제약에 의해 결정된다:

| 제약 요인 | 설명 | 1년 차 제약 강도 |
|---|---|---|
| **① 인지도 제약** | 타겟 소비자가 플랫폼의 존재를 아는가 | ★★★★★ (최대) |
| **② 기능 제약** | MVP가 SAM 전체를 커버하는 기능을 갖추는가 | ★★★★ (높음) |
| **③ 경쟁 제약** | 기존 플레이어(필리, 약들약, iHerb)로부터 점유율을 뺏을 수 있는가 | ★★★ (중간) |

1년 차에 가장 큰 제약은 **인지도**다. 아무리 우수한 제품을 출시해도, 소비자가 모르면 시장에 존재하지 않는 것과 같다.

### 1.2 SOM 산출 공식

> **SOM = 도달 가능 타겟 사용자 수 × 플랫폼 활성 사용률 × 사용자당 연간 수익(ARPU)**

각 변수를 1년 차 현실에 맞게 추정한다.

## 2. 1년 차 타겟 세그먼트 선정: 누구에게 집중하는가

### 2.1 선정 기준

1년 차에 자원이 제한된 상황에서, **모든 세그먼트**에 대응하는 것은 불가능하다. 아래 기준으로 1년 차 핵심 타겟을 선정한다:

| 기준 | 가중치 | 설명 |
|---|---|---|
| Pain Point 명확성 | 30% | 가치 제안이 즉각 체감되는가 |
| 자발적 유입 가능성 | 25% | 마케팅 없이도 찾아오는가 (검색 의도) |
| 커머스 전환율 | 25% | 수익으로 연결되는 속도 |
| 획득 비용 (CAC) | 20% | 한정된 예산으로 도달 가능한가 |

### 2.3 1년 차 핵심 타겟 결정

#### 🎯 Primary Target: Q1-A (가성비 최적화자)

| 항목 | 정의 |
|---|---|
| **누구** | 성분표를 읽을 수 있고, 함량 당 단가를 직접 계산하며, iHerb/쿠팡/네이버를 교차 비교하는 소비자 |
| **전체 규모** | 한국 건기식 소비자 1,500~2,000만 명 × 15~20%(Q1 비중) × 45~50%(Q1-A 비중) = **약 100~200만 명** |
| **1년 차 도달 가능** | 전체 Q1-A 중 3~5% = **약 3~10만 명** |
| **선정 이유** | Pain이 가장 구체적("채널 간 단가 비교를 자동화해줘"), 커머스 전환율 최고, MVP 기능(단가 계산기)이 이 세그먼트의 핵심 니즈와 직결 |

#### 🎯 Secondary Target: Q4-A (건강 계기 진입자)

| 항목 | 정의 |
|---|---|
| **누구** | 건강 검진 이상 소견 등으로 영양제를 처음 찾아보는 소비자. "의사가 비타민D가 부족하다고 했는데 뭘 사야 하지?" |
| **전체 규모** | 1,500~2,000만 명 × 25~30%(Q4 비중) × 35~40%(Q4-A 비중) = **약 130~240만 명** |
| **1년 차 도달 가능** | 전체 Q4-A 중 2~4% = **약 3~10만 명** |
| **선정 이유** | 검색 의도가 명확하여 SEO/콘텐츠 마케팅으로 자발적 유입 가능, 첫 구매 경험이 좋으면 Q1-A로 전환 → LTV 상승 |

#### 1년 차 총 도달 가능 사용자

> **Q1-A + Q4-A 합산: 약 6~20만 명** (보수적 ~ 기본 시나리오)

이 중 **월간 활성 사용자(MAU)**로 유지되는 비율을 30~50%로 가정하면:

> **1년 차 MAU: 약 2~10만 명** (12개월 차 기준 누적)

## 3. 1년 차 수익 모델별 SOM 산출

### 3.1 수익원 구조

1년 차에 현실적으로 가동 가능한 수익원과, 아직 가동이 어려운 수익원을 구분한다.

| 수익원 | 1년 차 가동 여부 | 이유 |
|---|---|---|
| **① 제휴 수수료 (커머스 연동)** | ✅ 가동 | MVP 시점부터 iHerb/쿠팡 파트너스 연동 가능 |
| **② 디스플레이·네이티브 광고** | ⚠️ 후반기 가동 | 최소 MAU 3~5만 이상 확보 후 광고주 유치 가능 |
| **③ 스폰서드 리뷰·인증 배지** | ❌ 미가동 | 플랫폼 신뢰도·영향력 미확보 상태에서 브랜드 설득 어려움 |
| **④ 프리미엄 데이터 (B2C 구독)** | ❌ 미가동 | 무료 사용자 기반이 충분히 쌓인 후 전환 가능 |
| **⑤ B2B 데이터·API** | ❌ 미가동 | 데이터 축적 최소 2년 이상 필요 |

### 3.2 수익원별 1년 차 추정

#### ① 제휴 수수료

| 변수 | 보수적 | 기본 | 비고 |
|---|---|---|---|
| 12개월 차 MAU | 20,000명 | 60,000명 | 누적 가입 대비 활성 비율 적용 |
| 월간 제휴 링크 클릭률 | 8% | 12% | Q1-A 높음, Q4-A 중간 |
| 클릭 후 구매 전환률 | 6% | 10% | iHerb 전환율 참조 |
| 건당 평균 구매액 | 35,000원 | 40,000원 | 건기식 평균 주문 단가 |
| 평균 수수료율 | 3% | 4% | iHerb 5~10%, 쿠팡 3%, 가중 평균 |
| **월간 수수료 수익** | **약 100만 원** | **약 1,150만 원** | |
| **연간 수수료 수익** | **약 700만 원** | **약 7,000만 원** | 초기 3개월은 거의 0, 점진 증가 |

> **산출 과정 (기본 시나리오):**
> 60,000 MAU × 12% CTR × 10% 전환 × 40,000원 × 4% = **약 1,152만 원/월** (12개월 차 기준)
>
> 그러나 1~3개월 차는 MAU가 5,000명 미만이므로, 연간 합산은 월간 최고치의 약 6개월분 수준인 **약 7,000만 원**으로 추정.

#### ② 디스플레이·네이티브 광고

| 변수 | 보수적 | 기본 | 비고 |
|---|---|---|---|
| 광고 가동 시점 | 10개월 차 | 7개월 차 | MAU 3만 이상 도달 시점 |
| 월간 페이지뷰 (PV) | 150,000 | 400,000 | MAU × 5~7 PV/session × 평균 세션 수 |
| CPM (1,000 PV당 광고 단가) | 3,000원 | 5,000원 | 건강 버티컬 미디어 CPM 참조 |
| 광고 가동 개월 수 | 3개월 | 6개월 | |
| **연간 광고 수익** | **약 135만 원** | **약 1,200만 원** | |

#### 1년 차 SOM 종합

| 수익원 | 보수적 | 기본 |
|---|---|---|
| 제휴 수수료 | 700만 원 | 7,000만 원 |
| 디스플레이 광고 | 135만 원 | 1,200만 원 |
| 기타 (스폰서드 등) | 0원 | 0원 |
| **1년 차 SOM 합계** | **약 835만 원** | **약 8,200만 원** |

> 보수적~기본 시나리오를 가중 평균하면, **1년 차 SOM은 약 0.5~1.0억 원** 범위로 추정된다.

### 3.3 SOM의 의미: 수익이 아닌 "증명"의 해

| 지표 | 값 | 의미 |
|---|---|---|
| 1년 차 SOM | **약 0.5~1.0억 원** | 3~4인 팀 인건비에 미달 |
| SAM 대비 SOM | 약 10~15% | 시장 내 초기 발판 확보 수준 |
| TAM 대비 SOM | 약 0.05~0.1% | 시장 전체의 극히 일부 |

이 수치가 시사하는 것은 명확하다: **1년 차에 수익으로 사업을 유지하려는 전략은 비현실적**이다. 1년 차의 목적은 수익 극대화가 아니라 다음 세 가지 전략적 자산의 확보에 있다:

1. **PMF 검증**: Q1-A와 Q4-A가 실제로 플랫폼을 반복 사용하는지 확인
2. **데이터 축적**: 제품·성분·가격 DB의 커버리지와 정확도 구축
3. **투자 유치 지표**: MAU, 리텐션율, NPS를 2년 차 투자 라운드의 근거로 확보

## 5. SOM 달성을 위한 핵심 전제 조건

### 5.1 반드시 충족해야 할 조건 (Killer Assumptions)

이 SOM 추정이 유효하려면, 아래 전제가 충족되어야 한다. 하나라도 실패하면 SOM은 하향 조정된다.

| # | 전제 조건 | 검증 방법 | 실패 시 영향 |
|---|---|---|---|
| 1 | **Q1-A의 핵심 pain ("수동 비교의 피로")이 진짜인가** | 베타 사용자 인터뷰 30건 + D7 리텐션율 25% 이상 | 리텐션 미달 시 타겟 재정의 필요 |
| 2 | **iHerb/쿠팡 제휴 프로그램이 안정적으로 유지되는가** | 제휴 수수료율 변동 모니터링, 최소 2개 채널 확보 | 수수료율 삭감 시 수익 50% 이상 하락 |
| 3 | **성분·가격 DB를 300개 이상 초기 구축할 수 있는가** | AI 기반 크롤링 + 수동 검증 파이프라인 | DB 부족 시 사용자 경험 저하 → 리텐션 하락 |
| 4 | **SEO로 Q4-A를 유기적으로 획득할 수 있는가** | "비타민D 추천" 등 핵심 키워드 상위 노출 3개월 내 | SEO 실패 시 유료 마케팅 의존 → CAC 상승 |
| 5 | **3~4인 팀으로 MVP 개발 + 운영이 가능한가** | 기술 스택 선정, AI 활용 개발 생산성 | 개발 지연 시 전체 로드맵 3~6개월 지연 |

### 5.2 주요 리스크 및 대응

| 리스크 | 발생 가능성 | 영향도 | 대응 방안 |
|---|---|---|---|
| **쿠팡 파트너스 수수료율 인하** | 중간 | 높음 | 수익원 다각화 (iHerb, 네이버 등 3개 이상 채널), 자체 광고 수익 비중 확대 |
| **필리(Pilly)의 가격 비교 기능 강화** | 높음 | 중간 | 차별화 포인트인 "함량 당 단가"와 "채널 간 교차 비교"에 집중, 필리가 커버하지 못하는 니치 공략 |
| **네이버/카카오의 건기식 정보 기능 내재화** | 낮음~중간 | 매우 높음 | 대형 플랫폼이 하지 않는 "심층 성분 분석" 영역 사수, 장기적으로 데이터 라이선싱 파트너십 전환 |
| **식약처 규제 강화 (비교 정보 = 광고 해석)** | 낮음 | 높음 | 법률 자문 사전 확보, "소비자 정보 제공"과 "광고"의 경계 명확화, 포지셔닝 조정 |
| **데이터 정확성 이슈 → 소비자 신뢰 훼손** | 중간 | 매우 높음 | 데이터 검증 프로세스 3중 구축 (AI 추출 → 자동 교차 검증 → 수동 샘플링), 오류 보고 체계 |

## 6. 2년 차 전환 시나리오: SOM이 SAM으로 확장되는 경로

1년 차 SOM은 그 자체로 사업을 유지하기 어렵지만, **2년 차 확장의 발판**으로서 가치가 있다. 1년 차에 확보한 자산이 2년 차에 어떻게 전환되는지를 명시한다.

### 6.1 1년 차 자산 → 2년 차 활용

| 1년 차 확보 자산 | 2년 차 활용 |
|---|---|
| MAU 6만 명 + 리텐션 데이터 | → 시드/프리A 투자 유치 (목표: 5~10억 원) |
| 제품 DB 1,000개 + 가격 이력 9개월 | → B2B 데이터 파일럿 (건기식 제조사 3~5개사) |
| SEO 콘텐츠 50편 + 검색 유입 채널 | → 유기적 트래픽 복리 성장, 광고 단가 상승 |
| Q4-A→Q1-A 전환 사례 | → PMF 검증 스토리 (투자자 피칭 핵심 자료) |
| 건기식 브랜드와의 광고 관계 | → 스폰서드 리뷰·인증 배지 수익원 추가 |

### 6.2 2년 차 SOM 예상 범위

| 구분 | 2년 차 예상 |
|---|---|
| MAU | 15~30만 명 |
| 수익원 | 제휴 수수료 + 광고 + 스폰서드 리뷰 + B2B 파일럿 |
| **연간 수익** | **약 3~8억 원** |
| SAM 대비 | 30~70% |

### 7.2 1년 차의 진짜 목표

| 순위 | 목표 | 측정 지표 | 성공 기준 |
|---|---|---|---|
| **1** | PMF 검증 | D30 리텐션율 | ≥ 15% |
| **2** | 사용자 기반 확보 | MAU (12개월 차) | ≥ 30,000명 |
| **3** | 데이터 자산 구축 | 제품 DB + 가격 이력 | ≥ 1,000개 제품, 9개월 이력 |

| **5** | 투자 유치 기반 마련 | 피칭 자료 완성도 | 핵심 지표 3개 이상 목표 달성 |

### 7.3 최종 판단

> **1년 차 SOM 0.5~1.0억 원은 독립 사업의 수익으로는 불충분하지만, 다음 두 가지 조건이 충족되면 전략적으로 유효한 출발점이다:**
>
> 1. **초기 자본(또는 부트스트랩 체력)**으로 최소 12~18개월의 런웨이를 확보할 것
> 2. **1년 차 KPI 달성을 근거로 2년 차 투자 유치에 성공할 것**
>
> 이 두 조건이 충족되면, 1년 차의 SOM은 단순한 매출 수치가 아니라 **"이 시장에서 우리의 가치 제안이 작동한다"는 증거**가 되며, 2~3년 차에 SAM의 30~70%를 점유하는 성장 경로의 출발점이 된다.
