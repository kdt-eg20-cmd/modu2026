# 📌 Value Proposition Sheet: 건강기능식품 비교 플랫폼  

**Executive Summary:**  
이 사업은 **“비교는 하되 최종 판단은 내릴 수 없는”** 건강보조식품 시장의 정보 비대칭 문제를 해결한다. 주요 고객 페르소나는 **C1(가성비 계산형)**, **C2(건강 계기 진입자)**, **A2(트렌드 추종형)**으로, 이들의 공통된 불만은 *“광고·전문용어 과잉으로 신뢰할 수 없는 정보 속에서 올바른 선택 기준을 찾기 어렵다”*는 점이다. 우리의 솔루션은 **“고객의 구매 판단을 대신 수행하는 결정 엔진”**이며, 이를 통해 고객의 탐색·계산 시간을 60분 → 5분 이내로 단축하고, 광고를 제거해 데이터 신뢰도를 높이며, 최적 장바구니를 제공하여 바로 구매로 연결한다.  

---

## 1. 페르소나 & 고객 여정 기반 핵심 문제 (Pain / Needs)  

- **공통 Pain:**  
  - **데이터 신뢰 부족:** 광고·마케팅 정보 범람으로 *“이 정보가 믿을 만한가?”*에 대한 불신.  
  - **전문성분 해석 난해:** 어려운 성분명·단위 때문에 올바른 비교 불가능.  
  - **판단 기준 부재:** *‘이 가격이 싸다면 적정한가?’*를 가늠할 근거 부족.  
  - **수동 비교 부담:** 엑셀 수기 계산에 **40–60분 소요** (AOS-DOS 분석에서 AOS 3.0) → 시간 낭비.  

- **페르소나별 Pain:**  
  - **C1 (가성비 최적화자, 한정훈):** 해외직구 할인 시 *환율·배송비 계산을 엑셀로 42분* 소요【user_files:5.JTBD.md†L293-L299】. → *“시간 낭비+계산 피로”*.  
  - **C2 (건강 계기 진입자, 박소연):** 건강 이슈 후 영양제 탐색, **성분지식 부족 + 광고 과다**로 45~90분 탐색에도 **결정 불확신**. → *“정보 과잉 속 판단 불가 → 불안”*.  
  - **A2 (트렌드 추종자, 정수빈):** SNS의 신제품 정보→충동구매 시도, **과학적 근거 미확인 + 가격 차이 이유 불명**으로 반복 후회. → *“팩트 부재 + FOMO 소비”*.  

위 데이터를 종합하면, 고객들이 최우선으로 해결해야 할 Pain은 **“광고 제거로 신뢰 확보 + 자동 단가 계산으로 시간 절감”**이다.

## 2. JTBD 기반 목표 (Goal / Job)  

- **Job 1 (C1):** *“환율·배송비·용량 등 모든 변수를 자동 반영해 **1알 당 최저 단가**를 바로 확인하고 결제하고 싶다.”*【user_files:5.JTBD.md†L293-L299】  
  - **측정 목표:** 탐색/계산 시간 **60분 → 5분 이하**, 오류율 **0%**.  

- **Job 2 (C2 + A2):** *“광고성 정보 없이 검증된 의학적 팩트만 보고 최종 결정을 내리고 싶다.”*【user_files:5.JTBD.md†L293-L299】  
  - **측정 목표:** 탐색 시간 **90분 → 10분 이하**, 결정 **확신도 100%**.  

- **Job 3 (신뢰 기반):** *“제공된 데이터를 내가 검증할 수 있어야 믿겠다.”*  
  - **측정 목표:** 데이터 출처 제공, 오류 신고 대응 (48시간 내 처리) 등 검증 프로세스 보장.  

각 Job은 앞서 정의된 페르소나의 요구와 연결된다(C1→Job1, C2/A2→Job2, 전 페르소나→Job3). 이상적인 결과는 *“고객이 30분 이내에 확신을 갖고 제품을 결정”*하는 상태이다.

## 5. 솔루션 핵심 제안 (Value Proposition)  

**구조:** *데이터 정제 ⇒ 비교 엔진 ⇒ 판단 지원 ⇒ 구매 연계*의 워크플로우를 완결하는 플랫폼.  
**핵심 기능:**  
- **1) 초자동화 단가 계산 엔진:** 여러 채널의 가격·환율·배송비·용량 정보를 합쳐 “1일 복용량(1정) 기준 원화 가격”을 실시간 계산. 👉 **엑셀 계산 삭제**(시간 60분→5초).  
- **2) 노이즈 캔슬링 팩트 대시보드:** 사용자 리뷰·광고 콘텐츠를 필터링하고, **성분 번역 + SCI/식약처 배지**만 표시. 👉 **광고 제거**로 신뢰 100% 확보.  
- **3) 최적 장바구니 자동생성:** 최저가 조합 채널별 비교 후 단추 클릭 한 번에 최적 장바구니 완성 (딥링크 제공). 👉 **구매 연결** 극대화.  

각 기능은 주요 Job에 대응(MVP 매핑 참조). 예: C1의 *“1일 단가 계산”*(F1), C2의 *“팩트 필터”*(F2) 등. 이를 통해 *“한 번의 클릭으로 최적 제품을 찾아 신뢰 기반으로 바로 결제”*가 가능해진다.

## 6. 차별적 가치 (Differentiation)  

| 요소             | 기존 서비스         | 우리 플랫폼                         |
|----------------|-------------------|----------------------------------|
| **가격 비교**      | 일부 가격 비교 기능    | 1일 단가 기준 **완전비교** (환율·배송 포함) |
| **성분 해석**      | 불가능 (전문가 필요)   | **자동 번역 + 전문 해설** 제공           |
| **판단 기준 제시**  | 없음               | **점수화/배지** 통한 객관적 판단 기준    |
| **투명한 신뢰 구조**  | 없음               | **데이터 출처 공개 + 오류 리워드** 보장   |
| **구매 연동**      | 제한적 (제휴광고 등)  | 최저가 장바구니 연동 & 쿠폰/적립 활용      |

- **“판단까지 지원”**: 모든 비교플랫폼과 다른 우리의 핵심 차별점은 *“구매 판단을 대신 제공”*하는 것.  
- **완전 투명성:** 데이터 출처(제조원·식약처 라벨) 공개로 *신뢰 100% 청정구역*을 제공.  
- **제조원 기반 비교:** 동일 제조원 제품을 통합 검색(숨은 동일품 제거)하며 함량당 단가 비교.  
- **커머스 완결성:** 비교에서 결제까지 이어주는 **풀 퍼널 지원**으로 사용자 이탈 차단.

## 9. 추가 권장 액션 (우선순위 1~3)  

1. **MVP 핵심 기능 개발 (F1·F2) 완수:** 단가 계산 엔진과 팩트 필터 먼저 개발. 조기에 내부·사용자 검증으로 시장 반응 확인.

## 가정 및 한계  

- **데이터 접근 가정:** 주요 커머스 사이트와의 스크래핑/제휴가 법적/기술적으로 가능해야 함. (예: 쿠팡 API 이용 가정)
