# 📋 Value Proposition 통합 문서

> **프로젝트:** RefineHealth (리파인헬스) — 건강기능식품 성분·가격 비교 의사결정 엔진  
> **작성 목적:** 시장 분석, 페르소나/CJM, JTBD 인터뷰, AOS-DOS 매트릭스, MVP 기획, 수익 모델, 기능 우선순위 맵 등 전체 사전 분석 결과물을 하나의 Value Proposition 프레임워크로 통합·정제한다.  
> **핵심 한줄 제안:** *"수동 엑셀 계산과 뒷광고 필터링에 지친 건기식 소비자들을 위한, 리얼-타임 1일 단가 계산 및 의학 팩트체크 플랫폼"*

## 1. 페르소나 및 CJM 기반 고객별 핵심 문제 서술 (Pain · Needs)

> 페르소나 정의 및 고객 여정지도(CJM) 분석 단계에서 도출된 핵심 문제를 페르소나별로 정리한다.

### 1-1. 공통 Pain Point (전 페르소나 관통)

| # | 공통 Pain | 설명 |
|---|---|---|
| P-공통1 | **데이터 신뢰 부족** | 광고·마케팅 정보 범람으로 비교 결과 자체를 신뢰할 수 없음 |
| P-공통2 | **성분 해석 장벽** | 전문 용어·단위(mg, IU, mcg 등) 이해 곤란 → 올바른 비교 자체가 불가능 |
| P-공통3 | **판단 기준 부재** | 가격이 '싸다/비싸다'를 가늠할 절대 기준선이 없어 구매 직전 불안 발생 |
| P-공통4 | **수동 비교의 시간·인지 비용** | 여러 채널 단가 계산에 엑셀 40~60분 소요 (AOS 3.0) → 극심한 시간 낭비와 피로 |

### 1-2. 페르소나별 Pain · Needs 상세

#### 🟡 핵심1. 한정훈 (36세, 가성비 최적화자 — "엑셀 환산러")

| 항목 | 내용 |
|---|---|
| **CJM 단계** | 탐색 → 비교 → 계산 → 구매 의사결정 |
| **Pain** | 여러 채널(iHerb, 쿠팡, 네이버 등)을 오가며 환율·용량·성분을 엑셀로 일일이 비교하고 단가를 계산하는 **극심한 반복 수동 탐색 피로도**. 해외직구 할인 행사 때 환율·배송비 수작업 계산에 **평균 42분** 소요 |
| **Needs** | 채널 변경이나 성분 비교 시 발생하는 시간 소모를 없애고 실시간으로 최저가 구매 타이밍을 포착하고 싶음. *"시간 낭비 + 계산 피로"* 해소가 최우선 |
| **CJM 이탈 지점** | 가격 판단 단계에서 **55~75%** 이탈 집중 — 성분 매트릭스 비교가 직관적이지 못하면 구매 자체를 포기 |

#### 🟢 핵심2. 박소연 (43세, 건강 계기 진입자 / 가족 건강 관리)

| 항목 | 내용 |
|---|---|
| **CJM 단계** | 건강검진 결과 수령 → 필요 성분 탐색 → 정보 과잉 혼란 → 판단 불능 → 포기 또는 불안 구매 |
| **Pain** | 건강검진 후 필요 성분을 찾으나 전문 용어 해석이 불가능하고 광고성 짙은 정보(블로그, 유튜버)로 인해 **극도의 불안과 혼란**. 45~90분 탐색에도 결정 불확신 |
| **Needs** | 복잡한 성분 지식 없이도 가족이 안심하고 섭취할 수 있는 '검증된 팩트'와 직관적인 추천. *"정보 과잉 속 판단 불가 → 불안"* 해소 |
| **CJM 이탈 지점** | 성분 비교 단계에서 전문 용어로 인한 인지 과부하 → 구매 퍼널 전체의 **55~75%** 이탈 |

#### 🔵 확장1. 정수빈 (27세, 트렌드 추종 탐색자)

| 항목 | 내용 |
|---|---|
| **CJM 단계** | SNS 트렌드 노출 → 충동 탐색 → 광고/팩트 구분 실패 → FOMO 구매 → 후회 |
| **Pain** | 인플루언서 마케팅과 상술에 속아 충동 구매(FOMO)를 반복한 쓰라린 경험. 진위 구분이 모호한 뒷광고에 대한 강한 불신. 과학적 근거 미확인 + 가격 차이 이유 불명 |
| **Needs** | 트렌디한 성분이 의학적으로 진짜인지 **단 5초 안에 판독**해주는 팩트체크 도구. 이를 쉽고 폼나게 친구들에게 공유하고 싶음. *"팩트 부재 + FOMO 소비"* 해소 |
| **CJM 이탈 지점** | 구매 후 후회 → 재탐색 루프 진입, 플랫폼 신뢰 자체가 형성되지 않음 |

### 1-3. 핵심 Pain 우선순위 (AOS-DOS 기반 정렬)

> AOS(Attractiveness of Opportunity Score)와 DOS(Difficulty of Satisfaction Score) 매트릭스에서 추출한 최우선 해결 Pain

| 우선순위 | Pain ID | Pain 내용 | AOS | DOS | 대상 페르소나 |
|---|---|---|---|---|---|
| **1st** | CORE-3 | 광고 범람에 대한 독립적 의학 정보 부재 | **4.0** | **3.6** | 핵심2, 확장1 |
| **2nd** | CORE-1 | 단가 수동 비교 방식의 극심한 피로 | **3.0** | **2.7** | 핵심1 |

## 2. JTBD 관점 인터뷰 결과에 따른 고객 상황별 목표 서술 (Goal · Job)

> JTBD 인터뷰 요약 카드에서 추출한 Job 선언(Job Statement)을 상황(Context) → 목표(Goal) → 측정 기준 구조로 정리한다.

### Job 1: 가성비 & 단가 최적화 Job (핵심1 — 한정훈)

| 항목 | 내용 |
|---|---|
| **상황 (When)** | 해외직구 할인·정기구매 시점이 도래하여 여러 채널의 가격을 비교해야 할 때 |
| **Job Statement** | 환율, 수량, 할인 조건이 모두 반영된 **정확한 '1일 복용량 단위 단가'를 5초 내로 자동 비교**하여, 불필요한 금전적·시간적 낭비 없이 최적 타이밍에 구매하려 한다 |
| **측정 목표** | 탐색/계산 시간 **60분 → 5분 이하**, 계산 오류 **0건** |
| **JTBD 인터뷰 원문** | *"엑셀로 달러 계산하다 현타온다"* — 실제로 소비자 대다수가 가격 비교에 40분 이상 소요 |

### Job 2: 안심 검증 & 팩트 확보 Job (핵심2 — 박소연 + 확장1 — 정수빈)

| 항목 | 내용 |
|---|---|
| **상황 (When)** | 제품 선택 시 광고성 정보가 넘쳐나지만 가족/본인의 건강을 위해 확실한 근거가 필요할 때 |
| **Job Statement** | 상업적 뒷광고를 전면 배제하고 **식약처/논문 등 객관적 의학 팩트로 필터링된 정보**만을 확보하여, 나의 구매 판단을 100% 확신하고 주변(가족/친구)에 자신 있게 공유하려 한다 |
| **측정 목표** | 탐색 시간 **90분 → 10분 이내**, 결정 확신도 **100%** |
| **JTBD 인터뷰 원문** | *"광고인지 사실인지 모르겠다"* — 핵심 페르소나 공통 불만 |

### Job 3: 데이터 신뢰 검증 Job (전 페르소나 공통)

| 항목 | 내용 |
|---|---|
| **상황 (When)** | 플랫폼이 제공하는 비교 데이터가 정말 정확한지 직접 확인하고 싶을 때 |
| **Job Statement** | 제공된 데이터를 **직접 검증 가능**해야 한다고 느낀다. 데이터의 출처와 원본을 투명하게 확인할 수 있어야 한다 |
| **측정 목표** | 원본 라벨 확인 가능, 오류 신고·수정 절차 확립 (48시간 내 수정 보장) |

## 3. 고객이 원하는 Outcome (측정 가능한 기대 효용)

> JTBD 분석 결과에서 고객이 달성하려는 이상적 상태를 측정 가능한 형태로 서술한다.

### 3-1. Outcome 정량 매트릭스

| Outcome | 현재 상태 (Before) | 목표 상태 (After) | 측정 지표 |
|---|---|---|---|
| **탐색 시간 압축** | 40~90분 (채널 교차 비교) | **5~10분** (원클릭 자동 비교) | TTC (Time-to-Conversion) |
| **단가 계산 시간** | 60분 (엑셀 수작업) | **5초** (자동화 엔진) | 계산 소요 시간 |
| **정보 신뢰도** | 낮음 (광고·마케팅 혼재) | **높음** (출처 투명 · 팩트 배지) | 신뢰도 설문 점수 |
| **구매 확신** | 불안·미확신 | **확신** (의학 근거 기반) | 결정 확신도 (%) |
| **구매 전환율** | 낮음 (이탈 55~75%) | **상승** (퍼널 전환 60%↑) | 비교 → 클릭 전환율 |
| **가짜 정보 식별** | 불가능 (자력 판별 한계) | **즉각 판독** (팩트 배지 3단계) | 팩트체크 정확도 |
| **SNS 공유 용이성** | 번거로움 (스크린샷·텍스트 수동 편집) | **1-Tap 카드 공유** (카카오톡 즉시 전송) | 세션당 공유 발생률 |

### 3-2. 이상적 최종 상태 (Ideal End-State)

> *"신뢰도 높은 단가 비교와 과학적 근거를 바탕으로, **30분 안에 확신을 갖고 제품을 결정**하고, 그 결론을 1-Tap으로 가족·친구에게 공유할 수 있는 상태"*

- **핵심1(한정훈):** 엑셀 없이 모든 채널의 1일 단가가 자동 정렬되어 최적 구매 타이밍을 놓치지 않음
- **핵심2(박소연):** 성분명을 몰라도 팩트 배지만 보고 가족에게 안심하고 권할 수 있음
- **확장1(정수빈):** 트렌드 성분의 진위를 5초 만에 확인하고, 팩트 카드를 친구에게 공유해 인정받음

## 4. 기존 대안 (Competitor / Substitute)

> 경쟁사 분석 및 시장 조사 결과를 종합하여 현재 고객이 사용 중인 기존 대안과 그 한계점을 체계적으로 정리한다.

### 4-1. 직접 경쟁 — 건강기능식품 정보 플랫폼

| 대안 | 대표 서비스 | 강점 | 한계 (미해결 Pain) |
|---|---|---|---|
| **개인화 추천 건강관리 앱** | 필라이즈, 필리, 핏타민, Carewith | 건강 설문 기반 맞춤 추천 제공 | 동일 성분 제품군 간의 **실질적 최저가/단가 교차 비교 기능 부재**. 판단 기준 불투명 |
| **이커머스 가격비교 포털** | 네이버쇼핑, 에누리, 쿠팡(로켓직구) | 수천 개 제품 가격 노출 | 제품별 1일 섭취량·함량 단위 미통일 → **성분 대비 진짜 가성비(단가) 직접 판별 불가** |
| **해외직구몰** | iHerb (2024년 매출 $2.4B, 사용자 1,240만 명) | 방대한 제품 라인업 | 구매 결정을 직접 돕지 않음. 성분 분석·판단 지원 없음 |

### 4-2. 간접 경쟁 — 정보 제공 채널

| 대안 | 형태 | 한계 (미해결 Pain) |
|---|---|---|
| **약사 유튜버·블로거** | 영상/텍스트 콘텐츠 | 겉으로는 객관적이나 실상은 *수익형 제휴 링크*·체험단 리뷰로 도배 → **정보의 독립성·객관성 완전 상실** |
| **온라인 커뮤니티** | 맘카페, 영양제 갤러리 등 | 정보 파편화, 검증 체계 부재, 개인 경험 기반 편향 |
| **약사 대면 상담** | 오프라인 약국 | 접근성 한정, 확장 불가, 가격 비교 기능 없음 |

### 4-3. 전통 브랜드 (레몬마켓 심화 주체)

| 대안 | 대표 사업자 | 한계 (미해결 Pain) |
|---|---|---|
| **전통 건기식 오프라인 브랜드** | 종근당, 정관장, 제약사 등 | 특허 원료 명목의 프리미엄 마케팅과 폐쇄적 유통 구조(제조원 비공개) → **정보 비대칭을 오히려 심화**, 레몬마켓 유도 |
| **OEM/ODM 제조사** | 노바렉스, 콜마 등 | 브랜드 신뢰는 있으나 개별 제품 비교·추천과 무관 |

### 4-4. 대체 행동 (Substitute Behavior)

| 대체 행동 | 소요 시간 | 한계 |
|---|---|---|
| 엑셀 수기 계산 | 40~60분/회 | 시간 낭비 + 계산 오류 리스크 |
| 블로그/유튜브 검색 | 45~90분/회 | 광고 오염, 팩트 미검증 |
| 지인·약사 구두 상담 | 가변적 | 비확장적, 데이터 기반 아님 |

> **핵심 결론:** 모든 기존 대안은 제품 '비교'까지만 돕고 **"최종 구매 판단"은 지원하지 않는다**. 소비자는 여전히 가격·성분에 대한 *절대 판단 기준*을 원하지만, 어디서도 제공받지 못하고 있다.

## 5. 우리 솔루션의 핵심 제안 (Value Proposition)

> JTBD와 Opportunity(AOS-DOS)를 통합적으로 고려하여 솔루션을 정의한다.

### 5-1. 핵심 제안 선언 (Value Proposition Statement)

> **"복잡한 영양제 스펙과 가격 구조를 직관적인 하나의 지표(1일 단위 최저 실단가)로 표준화하고, 철저한 팩트체크(광고 NO 배지)로 고객에게 '안심'과 '비교우위'를 돌려주는 시장 정제자(Market Refiner) AI 의사결정 엔진을 제공합니다."**

### 6-2. 핵심 차별화 포인트 3가지

#### ① "비교"를 넘어 "판단"까지 — 풀 파이프라인 완결

기존 서비스는 **데이터 정제 → 비교**까지만 지원하고 멈춘다. RefineHealth는 **데이터 정제 → 비교 → 판단 → 구매 연결**의 전 과정을 한 번에 완결하여 사용자 이탈을 구조적으로 차단한다.

```
기존: [검색] → [비교] → ❌ 판단 포기 (이탈 55~75%)
우리: [검색] → [자동 비교] → [팩트 판단] → [1-클릭 구매] → [공유]
```

#### ② 독립성의 구조적 보장 — "광고 수익 0원" 아키텍처

- 웹사이트 내 디스플레이 광고(배너) **구조적 전면 배제**
- 특정 브랜드 협찬 상위 노출(Sponsored) **기술적 불가능 설계**
- 수익은 오직 **제휴 CPA 수수료**(사용자 자발적 구매 시)와 **B2B 데이터 SaaS**로만 창출
- 이를 통해 "Anti-BS(반광고)" 포지셔닝의 신뢰를 장기적으로 수호

#### ③ 파편화된 데이터의 "표준화"와 "가공" — 진입 장벽이자 방어막(Moat)

- 상이한 함량 표기, 비표준 성분명, 채널별 묶음 수량, 실시간 환율 등 복잡한 매개변수를 **완전 분해 → 1일 실질 가격 통일**
- 상위 300~500개 주력 영양제 데이터 구축 → **Traffic Moat** 형성
- 축적 데이터 자체가 후발 주자의 모방을 어렵게 만드는 방어벽
