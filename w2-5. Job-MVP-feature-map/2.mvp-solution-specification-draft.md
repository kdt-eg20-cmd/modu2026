# MVP 개발 목표 수립 및 기획 아웃라인 초안 (Solution Specification)

> **작성 목적:** 지금까지 도출된 시장 분석, 페르소나, 여정지도, AOS-DOS, JTBD 인터뷰 결과를 총망라하여 실제 개발팀 및 기획팀이 착수할 수 있는 **'최소 기능 제품(MVP)의 핵심 스펙과 방향성'**을 정의함.
> **개발 철학:** "완벽한 앱"을 만드는 것이 아니라, 고객의 절박한 문제를 해결하는 "유일한 기능"을 시장에 가장 빨리 던져 검증하는 데 집중한다.

---

## 1. MVP 제품 비전 및 포지셔닝

*   **한 줄 정의 (Elevator Pitch):**
    👉 *"수동 엑셀 계산과 뒷광고 필터링에 지친 건기식 소비자들을 위한, 리얼-타임 1일 단가 계산 및 의학 팩트체크 플랫폼"*
*   **포지셔닝 선언 (Positioning Statement):**
    우리는 기존의 네이버/쿠팡 검색이나 영양제 갤러리와 달리, 판매 업체의 마케팅 노이즈를 100% 제거하고 오직 **1정당 찐 단가(Real Price)**와 **원천 데이터(식약처/논문 뱃지)**만을 제공하는 시장 내 유일한 **독립(Neutral) 플랫폼**으로 포지셔닝한다.

## 2. 타겟 오디언스 및 북극성 지표 (North Star Metric)

*   **Primary 타겟 (캐시카우):** 한정훈 (핵심1, 가성비 최적화자) — 엑셀 환산러
*   **Secondary 타겟 (그로스):** 박소연 (핵심2, 건강관리자) & 정수빈 (확장1, 트렌드 추종자) — 팩트체크 및 바이럴
*   **배제 타겟:** 극단1(디지털 소외), 비활성1(브랜드 맹신러) — MVP 단계 마케팅/기획 자원 투입 전면 금지
*   **북극성 지표 (North Star Metric):**
    **"탐색 시작 후 결제(또는 공유) 완료까지의 소요 시간 (TTC: Time-to-Conversion)"**
    *(목표: 기존 60분 → MVP 환경에서 5분 이내로 압축)*

## 3. MVP 핵심 개발 스펙 (Must-Have Features)

> ※ AOS-DOS 매트릭스 `🔥 Q1 혁신 기회` 및 `JTBD 인터뷰 결과`에 근거함.

### 🟡 Feature 1. 초자동화 환율·단가 환산 엔진 (Super-Calc Engine)
*(해결 Pain: C1의 1시간짜리 수동 엑셀 계산 과부하)*
*   **핵심 기능:** 해외 직구(달러/유로) 및 국내 채널(원) 간의 실시간 환율 적용.
*   **정규화 프로세스:** 60정, 120정, 분말(g) 등 상이한 제품 규격을 무조건 **"1일 복용량 단위 원화(₩) 가격"**으로 텍스트 파싱 및 수직 정렬.
*   **차별점:** 배송비와 기본 할인 코드가 적용된 최종가(Last Mile Price) 기준 랭킹 시스템.

### 🟢 Feature 2. 노이즈 캔슬링 팩트 대시보드 (Anti-BS Dashboard)
*(해결 Pain: 핵심2/A2의 정보 판단 불가 및 광고 피로도)*
*   **핵심 기능:** 상품 상세 내 모든 '사용자 리뷰', '별점', '블로그 홍보글' UI 원천 차단.
*   **대체 UI (의학 뱃지 시스템):**
    *   어려운 성분명을 일상어로 자동 번역 툴팁 제공 ("콜레칼시페롤" → "체내 흡수율이 높은 비타민 D3")
    *   논문 기반 팩트체크 결론을 3가지 신호등 뱃지로만 출력 (✅ 의학적 입증됨 / ⚠️ 근거 부족 / 🚫 과장 광고 주의)
*   **차별점:** 읽어야 할 텍스트를 최소화하고, "이것만 보면 된다"는 확신 부여.

### 🔵 Feature 3. 1-Tap 팩트 요약 카드 & 간편 공유 (Viral Engine)
*(해결 Pain: 외부 채널 공유의 번거로움, 극단1/비활성1 간접 유입 확보)*
*   **핵심 기능:** 검색-비교가 완료된 제품의 핵심 지표(1일 단가, 효능 뱃지, 최저가 링크)를 카카오톡 전용 썸네일 이미지 형태로 1초 만에 생성.
*   **차별점:** 앱 설치를 몰라도 링크 수신자는 웹뷰를 통해 **즉시 구매 결제창**으로 이동 가능 (극단1/비활성1의 진입 장벽 무력화).

### 🔴 Feature 4. 무결성 보장 시스템 (Data Trust System)
*(해결 Pain: 극단2 등 정보 비교 앱에 대한 카테고리 불신)*
*   **핵심 기능:** 가격 및 성분표 하단에 항상 **'식약처/제조사 원본 라벨 이미지 보러가기'** 아코디언 메뉴 탑재.
*   **장치:** UI 전면에 [오류 1건 제보 시 리워드 + 48시간 내 수정 보장] 정책 명시로 신뢰도 방어.

## 4. MVP 개발 범위에서 제외 (Out of Scope / Phase 2)

단기간 내 핵심 가설(초자동화 & 팩트체크) 검증에 집중하기 위해, 다음 항목은 과감히 제외합니다.
1.  **AI 개인화 맞춤 추천 (건강 설문 기반 추천):** 기술 부채가 크고 MVP 검증에 필수적이지 않음. (사용자가 검색하는 제품 비교부터 시작)
2.  **커뮤니티 및 자체 리뷰 작성 기능:** 광고가 유입될 여지를 제공하므로 초기 'Anti-BS' 포지셔닝에 치명적 독이 됨.
3.  **복잡한 헬스케어 온보딩 (이력 추적, 복약 알람):** 핵심1, 핵심2 타겟 모두 "구매" 전 탐색의 고통이 우선이지, "보조 앱"을 원하는 것이 아님.

## 5. 수익화 전략 및 비즈니스 모델 (Monetization)

본 플랫폼은 '독립적인 팩트 제공'이라는 훼손될 수 없는 가치를 지키기 위해, **웹사이트 내 디스플레이 광고(배너)나 특정 브랜드의 협찬 상위 노출(Sponsored)을 구조적으로 전면 배제**합니다. 대신 아래의 현실 데이터 기반 전환 수수료 및 B2B 모델을 채택합니다.

### 5.1 제휴 마케팅 수수료 (Affiliate CPA) — 1차 캐시카우
사용자가 비교 엔진을 통해 최종 결정한 '초저가 링크'를 클릭해 구매 시 발생하는 수수료(Affiliate)입니다.
*   **검증된 시장 데이터 기반 수익률 추정:**
    *   **객단가(AOV) 기준:** 한국건강기능식품협회의 2023-2024 실태조사 기준 타겟 가구의 연평균 건기식 지출액은 약 **34만 원** 수준. 타겟 페르소나의 제품 구매 단가를 역산하면 1회 평균 온라인 객단가(AOV)는 **40,000원 ~ 60,000원**으로 형성됨 (2~3개월치 묶음 구매가 대다수).
    *   **채널별 확정 수수료율 (2024년 기준 적용):**
        *   **iHerb (Rewards/Affiliate):** 기본 수수료 **5%** (신규 가입 유치 시 최대 10%). 해외 직구 가성비 타겟의 가장 강력한 수익 구간.
        *   **쿠팡 파트너스:** 플랫폼 정책 내 건강식품 구매 규정에 의한 기본 **3%** 지급 보장.
*   **예상 전환 가치:** 고객 1건 구매 완료 시 **평균 1,200원 ~ 3,000원** 안정적 순수익 발생. 핵심1 타겟의 재구매 라이프사이클(3~6개월)에 맞춘 반복 매출(Recurring Revenue)의 누적 모델.

### 5.2 마켓 인텔리전스 데이터 구독 (Market Intelligence B2B) — 중장기 스케일업
플랫폼에서 '마케팅 노이즈'를 제거했을 때 소비자가 특정 성분에 어떤 가격 저항선(Willingness to Pay)을 보이며 검색을 이탈하는지 기록한 하이엔드 로우-데이터(Raw Data)를 헬스케어 브랜드 및 건기식 제조사에 판매합니다.
*   **현실 기반 시장 동인:** 신규 진입하는 수많은 영양제 브랜드들은 '시장의 실제 성분당 원가 한계 기준선' 분석에 연간 수천만 원의 외부 비용을 선지출함. 
*   **BM 적용 모델:** 당사의 유저 행동 필터링 데이터를 기반으로 한 **B2B 대시보드 및 API 접근권**을 월 50만 원 ~ 150만 원 상당의 SaaS 구독 패키지로 제공하여, B2C 중립성은 지키되 소프트웨어적 고부가가치 마진을 확보합니다.

## 6. 성공 측정 기준 (MVP 검증 KPI)

MVP 런칭 후 핵심 가설이 검증되었는지 판단하는 기준선입니다.

| 구분 | 주요 KPI | MVP 1차 목표 수치 |
| :--- | :--- | :--- |
| **획득 (Acq)** | 건기식 / 영양제 검색어 SEO 노출 통한 오가닉 유입 | 월 방문자 10,000명 |
| **활성 (Act)** | 메인 화면 → 단가 산출 및 뱃지 화면으로의 퍼널 전환율 | 60% 이상 |
| **전환 (Rev)** | 비교 결과 화면 → '제휴사 구매처 링크(Affiliate)' 클릭률 | **15% 이상** (실제 클릭 발생 기반) |
| **바이럴 (Ref)** | 1세션 당 '카카오톡 공유하기 요약 카드' 발송 비율 | 생성 세션 10건당 1회 이상 (K-Factor 1.1) |
| **유지 (Ret)** | 영양제 재구매 주기 도래 시 (Day 30 ~ 60) 재방문율 | 20% 이상 (동일 유저의 전환율 지속) |
