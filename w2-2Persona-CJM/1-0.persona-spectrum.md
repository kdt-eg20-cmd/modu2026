|---|---|
| **페르소나 유형명** | 가성비 최적화자 (Q1-A) |
| **나이 / 직무** | 34세 / IT 스타트업 백엔드 개발자 |
| **역할** | 본인의 영양제 루틴을 스스로 설계·관리하는 자기주도형 소비자 |
| **사용 맥락** | iHerb·쿠팡·네이버를 오가며 비타민D, 오메가3, 마그네슘 3종을 3개월마다 재구매. 구매 때마다 40~60분 엑셀 작업으로 단가 비교. |

| **대체 솔루션** | 개인 구글 시트에 함량 당 단가 계산식 직접 구현, 쿠팡 가격 추적 크롬 확장 프로그램(비건기식 특화 아님), iHerb 즐겨찾기 목록 수동 관리 |

| **사용 맥락** | 겨울철 건강 검진에서 비타민D 결핍 판정. 약사에게 "드시면 좋아요"라는 말만 듣고 퇴장. 네이버에서 "비타민D 추천" 검색 후 블로그 7개, 유튜브 3개 시청했으나 제품이 30개 이상 등장해 오히려 혼란. |

| **대체 솔루션** | 네이버 베스트셀러 상위 제품 구매, 맘카페 지인 추천 의존, 약국에서 약사 상담(비용 없음) |

### Core-3. 박지현 | "우리 아이 영양제, 이게 진짜 맞는 건지 불안해"

| 항목 | 내용 |

| **사용 맥락** | 자녀용 아연 제품 구매 시 맘카페 검색 → 브랜드 논쟁 → 60개 댓글 독파 → 결론 없음. 성인용과 어린이용의 함량 차이, 약과의 상호작용 여부를 확인하고 싶으나 통합된 정보가 없다. |

| **목표** | 가족 구성원별(어린이·성인·노인)로 맞춤화된 안전 정보와 용량 가이드를 한 플랫폼에서 확인하고, 커뮤니티 경험담이 아닌 독립적 정보에 기반해 결정을 내리고 싶다. |
| **감정** | 잘못 선택했을 때의 책임감 불안이 지배적. 정보를 찾을수록 판단이 어려워지는 좌절감. 검증된 정보를 제공받으면 강한 로열티 형성. |
| **대체 솔루션** | 맘카페 육아 커뮤니티 추천 의존, 소아과 의사에게 질문, 약국 방문 상담 |

| **감정** | 비교 과정 자체를 즐기지만, 수동 노동의 반복에는 피로감. "이 정도 과정을 자동화해 주는 서비스가 있다면 당장 쓸 것"이라는 기대감이 강함. |
| **대체 솔루션** | 다나와(비건기식 전문 아님), 직접 구성한 가격 비교 노션 DB, iHerb 위시리스트 수동 체크 |

### Adjacent-1. 손지영 | "같은 거 재구매하다 갑자기 성분이 바뀌었더라고요"

| 항목 | 내용 |
|---|---|
| **페르소나 유형명** | 효율적 전문가 / 리피터 (Q2) |
| **나이 / 직무** | 36세 / 병원 소속 영양사 |
| **역할** | 성분 지식이 풍부하지만, 이미 검증된 제품을 반복 구매하는 패턴. 탐색 시간을 거의 투자하지 않음. |
| **사용 맥락** | 본인이 복용하는 영양제 5~6종을 2~3년째 같은 브랜드에서 주기 구매. 새로운 제품을 비교할 동기는 낮지만, 가끔 "더 좋은 제품이 나왔는지 체크하고 싶다"는 생각이 든다. |
| **주요 문제** | 기존 구매 제품의 가격이 오르거나 성분 함량이 조용히 변경되어도 인지하지 못한다. 바쁜 직무 특성상 탐색에 시간을 쓸 여유가 없어 '수동적 고착'이 이어진다. |
| **목표** | 알아서 찾아오는 서비스: 복용 중인 제품의 가격·성분 변화를 자동으로 감지하고, 동일 성분의 더 나은 대안이 나오면 알림을 받고 싶다. |
| **감정** | 현재 상태에 대한 표면적 만족감이나 내심 "놓치고 있는 것이 있지 않을까" 하는 잠재 불안. 정보가 찾아오지 않으면 굳이 찾아가지 않는 수동성. |
| **대체 솔루션** | 기존 제품 구독 서비스 자동 결제, 동료 영양사들 간 비공식 정보 공유 |

| **목표** | 스택 빌더 기능: 복수 제품을 담으면 총 성분 프로파일, 중복·부족, 성분 간 상호작용, 총 비용을 자동 분석해 주는 도구를 원한다. |
| **감정** | 지식이 높을수록 더 세밀하게 비교하고 싶은 욕구가 강함. 수동 계산의 한계에서 오는 답답함. 콘텐츠의 깊이가 있는 플랫폼에 강한 신뢰감을 느낌. |
| **대체 솔루션** | Examine.com (영어, 제품 비교 기능 없음), 개인 노션 트래커, 트레이너 커뮤니티 비공개 공유 |

| **나이 / 직무** | 26세 / 대기업 마케팅 직군 신입사원 |

| **사용 맥락** | 필리(Pilly) 앱에서 복용 알림 기능을 쓰고 있으나, 제품 비교 기능이 미흡해 추가 탐색 시 네이버·블로그를 병행한다. 제품 간 함량 당 단가를 앱 내에서 비교할 수 없어 결국 별도 검색 필요. |
| **주요 문제** | 기존 사용 앱(필리 등)이 복용 알림·일정 관리는 지원하지만, 성분 비교 + 가격 비교 깊이가 부족해 구매 결정을 내릴 때마다 앱 밖으로 이탈해야 한다. |
| **목표** | 복용 관리 + 성분 비교 + 가격 비교가 하나의 플랫폼에서 끊김 없이 연결되길 원한다. 비교 결과에서 바로 구매 링크로 이어지는 동선을 원한다. |
| **감정** | 기존 앱에 대한 불완전한 만족감. "이 기능만 있으면 완벽한데"라는 니즈가 명확. 더 좋은 대안을 발견하면 전환 장벽이 낮음. |
| **대체 솔루션** | 필리(Pilly) 복용 알림 앱, 네이버 쇼핑 비교, 개인 메모앱에 수기 정리 |

## PART 3. 극단 사용자 (Extreme Users) — 2명

> **정의:** 가장 자주 실패를 경험하거나 극단적 제약 상황에 처한 사용자. 제품의 경계와 개선점을 드러내는 역할.

---

### Extreme-1. 임도현 | "영양제 10종 먹다가 부작용 생겼던 사람"

| 항목 | 내용 |
|---|---|
| **페르소나 유형명** | 과잉 복용 실패 경험자 (Q1-B 고착 후 부작용 경험) |
| **나이 / 직무** | 42세 / 스타트업 대표 |
| **역할** | 건강 관리 의지가 극도로 높아 영양제를 통한 '바이오해킹'을 시도했으나 정보 관리 실패로 부작용 경험 |
| **사용 맥락** | 2년간 개인적으로 구성한 영양제 스택이 10종에 달했음. 특정 시기에 피로감과 소화 이상 발생. 사후에 비타민A 복합 섭취로 인한 과잉 섭취를 의심했으나, 당시에는 총 성분 합산을 확인할 방법이 없었음. |
| **주요 문제** | 복수 제품의 성분 중복으로 인한 과잉 섭취 위험을 사전에 감지할 수 없었다. 부작용 발생 후 원인 규명을 도와줄 플랫폼도 없었다. 영어 자료 외에는 성분 간 상호작용 정보를 국내에서 구하기 어려웠다. |
| **목표** | 복용 스택 전체의 총 성분 합산량과 상한 기준(UL, Tolerable Upper Intake Level)을 자동 비교하는 안전 체크 기능. 성분 간 충돌·경쟁 흡수 경고 기능. |
| **감정** | 부작용 경험 후 의심과 불신이 높아짐. 정보에 대한 높은 기준과 검증 욕구. 한 번 실패했기 때문에 "이 플랫폼이 틀리면 더 큰 피해"라는 리스크 인식이 높음. |
| **대체 솔루션** | 기능의학과 전문의 상담(비용 높음), Examine.com + 논문 직접 검색, 영양 전문가 유료 1:1 컨설팅 |
| **플랫폼 설계 시사점** | 총 섭취량 자동 합산 + 안전 상한 경고 기능 필수. 성분 상호작용 데이터베이스. 이 사용자를 만족시키는 기능이 모든 스택 사용자의 안전망이 됨. |

---

### Extreme-2. 권미영 | "영양제 샀다가 환불 세 번 한 사람"

| 항목 | 내용 |
|---|---|
| **페르소나 유형명** | 반복 구매 실패자 / 정보 불신 극단 사례 (Q4-A 실패 패턴) |
| **나이 / 직무** | 52세 / 고등학교 국어 교사 |
| **역할** | 건강 관심이 높아 여러 번 영양제를 구매했으나 매번 다른 이유(맞지 않음, 속 불편함, 함량이 너무 적음 등)로 환불하거나 복용 중단함 |
| **사용 맥락** | 폐경 전후 영양제를 찾으며 인터넷 검색 → 블로그 → 유튜브를 반복했으나 "이건 광고인지 추천인지 모르겠다"는 혼란이 3년 이상 지속. 구매 영양제 총 5종, 그 중 지속 복용 성공 0종. |
| **주요 문제** | 광고성 콘텐츠와 실제 정보를 구분할 수 없다. 구매 결정 기준이 없어 매번 "많이 팔리는 거"를 선택했다가 실패한다. 성분 정보를 이해하려 해도 용어가 어렵고, 결국 마케팅 문구에 끌린다. 나이·증상에 맞는 제품이 무엇인지 통합 가이드가 없다. |
| **목표** | 나의 나이, 성별, 증상(폐경 전후 피로·수면)을 입력하면 광고 없이 객관적으로 추천되는 제품 리스트를 신뢰할 수 있는 근거와 함께 받고 싶다. |
| **감정** | 반복 실패로 인한 깊은 불신. "어차피 다 광고겠지"라는 냉소. 그러나 권위 있는 정보원(의사, 식약처 인용 등)에는 여전히 반응. 성공 경험이 1회라도 생기면 강한 전도사가 될 수 있음. |
| **대체 솔루션** | 약국 방문 약사 추천(건별), TV 홈쇼핑 구매, 지인 입소문 |
| **플랫폼 설계 시사점** | 광고 여부 명확 표시, 증상/연령 기반 진입 흐름, 단순화된 판단 기준(A~F 등급 등)이 이 사용자를 회복시키는 핵심. 첫 성공 경험 설계가 최우선. |

## PART 4. 비활성 사용자 (Non-users) — 2명

> **정의:** 이 플랫폼을 현재 사용하지 않거나 적극적으로 회피하는 집단. 진입 장벽과 거부 요인을 파악하는 역할.

---

### Non-user-1. 황순자 | "우리 때는 종근당이면 다 됐는데"

| 항목 | 내용 |
|---|---|
| **페르소나 유형명** | 브랜드 의존 수동적 수용자 (Q3) |
| **나이 / 직무** | 63세 / 전업주부 (자녀 출가 후 독립 생활) |
| **역할** | 30년 이상 특정 대형 브랜드(종근당비타민, 광동 쌍화탕 등)를 습관적으로 구매해온 비교 회피 소비자 |
| **사용 맥락** | TV 광고에서 본 제품, 홈쇼핑에서 구매한 제품, 자녀가 사다 준 제품을 순서대로 복용. 성분표를 본 적 없고 볼 이유도 모른다. 스마트폰은 쓰지만 앱 설치에 적극적이지 않음. |
| **주요 문제 (비사용 이유)** | 성분 비교 필요성 자체를 인식하지 못한다. "비싸면 좋은 거지, 싼 게 뭔 소용이야"라는 인식. 새로운 앱을 배우는 것에 대한 거부감(디지털 부담감). 본인이 정보 피해자라는 인식 전무. |
| **진입 장벽** | 서비스 인식 부재, 디지털 리터러시 장벽, 브랜드 신뢰에 기반한 현재 만족감, 변화에 대한 저항 |
| **감정** | 현재 상태에 대한 안정감과 만족감. "굳이 복잡하게 찾아보고 싶지 않다"는 단순화 욕구. 멀리서 작동하는 플랫폼(자녀가 공유한 비교 카드)에는 간접 반응 가능. |
| **대체 솔루션** | TV 홈쇼핑, 대형 마트 진열대, 자녀 선물, 지인 추천 |
| **플랫폼 접근 전략** | 직접 유입 불가. Q4-B 사용자(자녀)가 공유하는 비교 카드와 SNS 바이럴을 통해 간접 노출. 장기적으로 '인쇄용 비교 요약'이나 '가족 공유 기능'으로 접근 가능. |

### Non-user-2. 박성호 | "성분 비교 플랫폼? 어차피 광고 아닌가?"

| 항목 | 내용 |
|---|---|
| **페르소나 유형명** | 정보 냉소주의자 / 신뢰 결여 비활성 사용자 |
| **나이 / 직무** | 44세 / 공무원 (시청 행정직) |
| **역할** | 디지털 플랫폼과 온라인 정보에 대한 신뢰가 낮아, 의도적으로 비교 플랫폼을 피하는 회의주의자 |
| **사용 맥락** | 과거 특정 가격비교 사이트에서 "최저가"로 표시된 제품이 실제로는 제휴 광고임을 뒤늦게 알고 불쾌한 경험. 건기식도 "어차피 제조사 돈 받고 추천하는 거겠지"라는 인식. 직접 약사 또는 가정의학과를 신뢰함. |
| **주요 문제 (비사용 이유)** | 플랫폼의 수익 구조(제휴 수수료)가 곧 '이해 충돌'이라고 인식. 독립적 정보는 광고를 붙이는 순간 신뢰도가 소멸한다고 봄. 어떤 플랫폼도 "완전 중립"이 될 수 없다고 믿음. |
| **진입 장벽** | 플랫폼의 수익 구조에 대한 근본적 불신, 제휴 광고 = 편파 정보라는 고정 인식, 비교 자체보다 전문가 상담을 선호, 디지털 정보에 대한 회의적 태도 |
| **감정** | 냉소와 의심이 기본값. 그러나 투명한 수익 구조 공개(예: 제휴 마크 표시, 광고·비광고 명확 구분)에는 부분적으로 반응 가능. 공공기관 인증이나 식약처 데이터 기반임을 명시하면 신뢰 반응 증가. |
| **대체 솔루션** | 약국 약사 상담, 가정의학과 방문, 소비자원 리포트 열람, 오프라인 직구 지인 추천 |
| **플랫폼 접근 전략** | 수익 구조 투명성 공개(제휴 표시 의무화 + 비광고 영역 명확화). 식약처·학회 데이터 직접 연동을 신뢰 앵커로 활용. 이 세그먼트를 완전히 설득하는 것보다, 불신 요인 최소화로 '거부감 철회' 유도가 현실적. |
