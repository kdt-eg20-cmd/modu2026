## CARD 01 · 핵심 사용자 (Core)

```
┌─────────────────────────────────────────────────────────────┐
│  🔵 CORE USER                                               │
│  "같은 비타민D인데, 지금 제일 싼 데가 어딘지 5초 안에 알고 싶다"        │
└─────────────────────────────────────────────────────────────┘
```

### 한정훈 · 36세 · IT기업 백엔드 개발자

| | |
|---|---|
| **세그먼트** | Q1-A 가성비 최적화자 |
| **성분 리터러시** | ★★★★★ 높음 — 성분표를 스스로 해석하고 단위 환산도 직접 계산 |
| **탐색 강도** | ★★★★★ 높음 — 구매마다 40~60분, 정기적·반복적 비교 행동 |

#### 🔴 Problem — 그가 겪는 문제

> *"iHerb는 달러, 쿠팡은 원화, 네이버는 또 다른 가격. 같은 비타민D 1,000IU를 진짜로 비교하려면 환율 적용, 용량별 나눗셈, 할인 반영까지 해야 하는데, 이걸 전부 스프레드시트에 손으로 입력하고 있어요."*

- 탭 8개를 열고 각 채널 가격을 교차 대조하는 데 **건당 40~60분** 소요
- 동일 제품이라도 60정·120정·240정 용량에 따라 단가가 모두 다름 — **매번 나눗셈 반복**
- 할인·쿠폰·적립금 적용 후 실질 가격 계산까지 더하면 비교 복잡도가 **기하급수적으로 증가**
- 가격이 내려가는 타이밍을 놓쳐 **최적 구매 시점 선택 실패**

#### 🎯 Goal — 그가 원하는 것

> *"비타민D 1,000IU 기준, 지금 이 순간 iHerb·쿠팡·네이버 중 어디가 제일 싼지 5초 안에 보여줬으면 좋겠어요. 거기서 바로 구매 링크로 넘어가고요."*

- **채널 통합 단가 자동 비교** (환율 실시간 반영)
- **용량 단위 자동 통일** (1일 복용 기준 환산)
- **가격 하락 알림** (특정 제품·성분 지정)
- 탐색 시간 60분 → **5분 이하**로 단축

#### 📍 Context — 플랫폼을 사용하는 순간

| 트리거 | 행동 | 이탈 원인 (현재) |
|---|---|---|
| iHerb 세일 시작 알림 수신 | 채널 간 즉시 가격 비교 필요 | 환율·용량 계산을 직접 해야 함 |
| 2~3개월 정기 재구매 주기 도래 | 최저가 채널 선택 | 지난번 가격과의 비교 수단 없음 |
| 신제품 정보 커뮤니티 접수 | 기존 복용 제품과 비교 | 새 제품 성분값을 어디서 가져올지 모름 |

#### 💬 Emotion — 감정 곡선

```
탐색 시작 ──► 비교 중 ──────────────► 구매 결정
   "또 해야 해"    "탭 몇 개짜리야..."        "됐다. 근데 다음엔 자동화 도구 써야지."
   [피로] [체념]      [집중, 짜증]            [해결감 + 잠재 불만 잔존]
```

#### 🛠 대체 솔루션 (현재)
- 개인 구글 스프레드시트 (환율 계산식 직접 작성)
- iHerb 내 정렬 기능 (채널 간 비교 불가)
- 에누리 건강플러스 (건기식 전문 비교 기능 미흡)

#### ✅ 플랫폼 적용 포인트

| 기능 | C1에게 의미하는 것 |
|---|---|
| 채널 간 단가 자동 비교 (환율 실시간) | "드디어 스프레드시트가 필요 없어진다" |
| 용량별 단가 자동 통일 (1일 기준) | "내가 계산기 두드릴 필요가 없어진다" |
| 가격 하락 Push 알림 | "지금이 살 때다"를 플랫폼이 알려준다 |
| 커머스 링크 직접 연결 | 결정 후 이탈 없이 구매까지 완결 |

> 💡 **PMF 지표:** C1의 제휴 링크 CTR ≥ 20% → 커머스 연동 모델 유효성 확인

## CARD 02 · 확장 사용자 (Adjacent)

```
┌─────────────────────────────────────────────────────────────┐
│  🟢 ADJACENT USER                                           │
│  "이게 진짜 효과 있는 건지, 이 가격이 맞는 건지 팩트체크하고 싶어"       │
└─────────────────────────────────────────────────────────────┘
```

### 정수빈 · 27세 · 뷰티 브랜드 마케터

| | |
|---|---|
| **세그먼트** | Q4-C 트렌드 추종 탐색자 |
| **성분 리터러시** | ★★ 낮음 — 성분명을 알아도 함량의 의미는 모름 |
| **탐색 강도** | ★★★ 중간 — 트렌드 계기일 때만 탐색, 불규칙 |

#### 🔴 Problem — 그녀가 겪는 문제

> *"인스타에서 글루타치온이 핫하다길래 검색했어요. 근데 다 광고 같고, 같은 500mg인데 1만원짜리도 있고 8만원짜리도 있어요. 뭐가 다른 건지 모르겠어서 그냥 인플루언서 쓰는 거 샀다가 후회했어요."*

- 인플루언서 추천이 **스폰서 광고인지 진짜 추천인지 구분 불가**
- 동일 성분·함량인데 **가격 8배 차이**의 근거를 알 수 없음
- "효과가 입증됐다" vs. "근거 부족"이라는 **상반된 정보의 혼재**
- FOMO(놓칠까봐 불안)에 의해 **충동 구매 → 후회 사이클** 반복

#### 🎯 Goal — 그녀가 원하는 것

> *"이 성분이 실제로 근거가 있는 건지, 비싼 게 진짜 좋은 건지 아니면 마케팅비인지를 객관적으로 빠르게 알고 싶어요. 그리고 친구들한테 공유도 하고 싶고요."*

- 트렌드 성분의 **과학적 근거 수준 팩트체크** (근거 있음 / 제한적 / 미비)
- 가격 차이가 **품질 차이인지 마케팅비 차이인지** 판단 지표
- 광고·비광고 콘텐츠 **명확 구분 표시**
- SNS 공유용 **비교 결과 카드 자동 생성**

#### 📍 Context — 플랫폼을 사용하는 순간

| 트리거 | 행동 | 이탈 원인 (현재) |
|---|---|---|
| 인스타·유튜브에서 트렌드 성분 발견 | "이게 진짜야?" 즉시 검색 | 광고성 콘텐츠 도배로 객관 정보 없음 |
| 인플루언서 할인코드 게시물 노출 | FOMO로 구매 충동 | 가격 합리성 판단 수단 없어 충동 결정 |
| 구매 후 성분 확인 시도 | "내가 산 게 맞는 건가?" 재탐색 | 이미 구매 완료 후 → 후회만 남음 |

#### 💬 Emotion — 감정 곡선

```
SNS 발견 ──► 검색 ──────► 구매 결정 ──► 구매 후
"이거 나도?" [FOMO]  "다 광고야..." [의심]  "그냥 샀다" [타협]  "왜 샀지" [후회]
```

#### 🛠 대체 솔루션 (현재)
- 인스타·유튜브 인플루언서 콘텐츠 (광고·추천 혼재)
- 네이버 블로그 검색 (체험단 리뷰 중심)
- 올리브영 매장 직원 추천 (성분 깊이 없음)

#### ✅ 플랫폼 적용 포인트

| 기능 | A2에게 의미하는 것 |
|---|---|
| 트렌드 성분 근거 등급 표시 | "이게 진짜 효과 있는 건지" 5초 팩트체크 |
| 광고 성분 / 비광고 독립 평가 분리 표시 | "이 추천이 광고인지 아닌지" 즉시 구분 |
| 동일 성분 가격 분포 시각화 | "8만원짜리가 1만원짜리보다 7배 좋은 건 아니다" 납득 |
| SNS 공유용 비교 카드 1클릭 생성 | "친구들한테 바로 공유" → 바이럴 유입 엔진 |

> 💡 **핵심 전략:** A2가 공유한 비교 카드가 핵심2(박소연)의 첫 플랫폼 유입 경로가 됨 → 확장 사용자가 핵심 사용자를 끌어오는 바이럴 구조

## CARD 03 · 극단 사용자 (Extreme)

```
┌─────────────────────────────────────────────────────────────┐
│  🔴 EXTREME USER                                            │
│  "'어머니, 이거 드세요' 딱 하나만 말해줬으면 좋겠어요"               │
└─────────────────────────────────────────────────────────────┘
```

### 나경아 · 62세 · 은퇴 교사

| | |
|---|---|
| **세그먼트** | Q4-A 극단 변형 — 디지털 리터러시 제약 |
| **성분 리터러시** | ★ 매우 낮음 — 성분표를 볼 줄 모름. "비싸면 좋다" 인식 |
| **탐색 강도** | ★★ 낮음 — 건강 위기 계기 시 간헐적 시도, 곧 포기 |

#### 🔴 Problem — 그녀가 겪는 문제

> *"당뇨 판정 받고 영양제 알아보려 했는데... 앱 설치가 먼저 안 되고, 겨우 들어가도 글씨가 작고 버튼이 어딨는지 모르겠어서 그냥 닫았어요. 결국 홈쇼핑에서 샀어요."*

- 스마트폰 **앱 설치·로그인 자체가 첫 번째 장벽**
- 접속해도 **작은 글씨·복잡한 필터·메뉴 구조**에서 이탈
- "성분 비교"가 무엇인지 **개념 자체가 낯선 진입 전**의 문제
- 결국 **TV 홈쇼핑·자녀 대리 구매**에 의존 → 정보 격차 지속

#### 🎯 Goal — 그녀가 원하는 것

> *"뭘 먹으면 되는지 딱 하나만 알려줬으면 좋겠어요. 광고 아닌 걸로."*

- **단 하나의 결론** — "이 제품 드세요" (비교가 아닌 결론 제시)
- **선택지 최소화** — 3개 이하, 큰 글씨, 핵심 정보만
- **자녀 공유 가능 포맷** — 카카오톡으로 받은 링크 하나로 바로 구매

#### 📍 Context — 플랫폼을 '간접적으로' 사용하는 순간

| 트리거 | 행동 | 실제 흐름 |
|---|---|---|
| 당뇨·혈압 등 건강 위기 계기 | 스스로 검색 시도 → 인터페이스 장벽 → 포기 | TV 홈쇼핑으로 전환 |
| 자녀(C2형)가 카카오톡으로 링크 공유 | 링크 클릭 → 큰 글씨·단순 결론 페이지 | 자녀에게 "이거 사줘" 요청 |
| 지인이 "이 영양제 좋대" 구전 | 확인하려 하지만 방법 모름 | 지인 추천 그대로 구매 |

#### 💬 Emotion — 감정 곡선

```
건강 위기 ──► 시도 ──────────► 포기 ──────────► 의존
"뭐 먹어야 하지?" [절박]  "모르겠어" [좌절·소외]  "나는 못 해" [무력감]  홈쇼핑·자녀 [체념]
```

#### 🛠 대체 솔루션 (현재)
- TV·라디오 홈쇼핑 (스토리텔링에 끌려 구매)
- 약국 약사 대면 상담 (이동 비용 발생)
- 자녀·지인 대리 구매 (본인 의사결정 불가)

#### ✅ 플랫폼 적용 포인트 (접근성 설계 기준)

| 설계 기준 | E1이 통과해야 하는 테스트 |
|---|---|
| **3탭 이내 단일 결론 UX** | 처음 방문 후 3번의 클릭으로 "이 제품 드세요" 화면에 도달하는가? |
| **16px 이상 글씨 크기·고대비 UI** | 안경 없이도 핵심 정보가 보이는가? |
| **카카오톡 공유 1탭** | 자녀가 어머니에게 결론 링크를 단 1번 터치로 공유할 수 있는가? |
| **증상 기반 진입 ("당뇨에 좋은 영양제")** | 성분명 없이 증상 단어로 검색해도 결론에 도달하는가? |

> 💡 **설계 원칙:** E1을 만족시키는 단순함이 핵심2(박소연)의 첫 진입 허들도 낮춘다 — 극단 사용자 기준이 핵심 사용자 UX의 하한선

## CARD 04 · 비활성 사용자 (Non-user)

```
┌─────────────────────────────────────────────────────────────┐
│  ⚫ NON-USER                                                │
│  "종근당이면 됐지. 뭘 또 비교해요?"                             │
└─────────────────────────────────────────────────────────────┘
```

### 조미라 · 58세 · 아파트 관리사무소 경리

| | |
|---|---|
| **세그먼트** | Q3 수동적 수용자 — 브랜드 의존형 |
| **성분 리터러시** | ★ 없음 — 성분표를 볼 생각 자체가 없음 |
| **탐색 강도** | ★ 없음 — 탐색 없이 반사적 재구매 |

#### 🔴 Problem — 그녀가 인식하지 못하는 문제

> *"20년 넘게 종근당 먹고 있는데 뭐가 문제예요? 비싸면 좋은 거 아닌가요?"*

**⚠️ 이 페르소나의 핵심 특징: Pain을 인식하지 못하는 것이 Pain이다**

- 동일 성분·함량 제품이 **3분의 1 가격에 존재**한다는 사실을 모른다
- "비싼 게 좋은 것"이라는 **가격-품질 오인**이 고착되어 있다
- 성분 비교 플랫폼이 존재해도 **방문할 이유 자체가 없다**
- 정보를 제시하면 오히려 **"내가 바보라는 거냐"** 는 방어 반응이 유발된다

#### 🎯 Goal — 그녀의 목표

> (목표 부재) *"지금 먹는 거 만족하니까 건드리지 마세요."*

- 플랫폼을 **자발적으로 찾아올 의도가 없다**
- 다만 **가격 인상 체감, 자녀의 개입, 건강 뉴스** 등 외부 계기 시 Q4로 이동 가능

#### 📍 Context — 플랫폼과 '간접 접점'이 생기는 순간

| 트리거 | 가능한 간접 접점 | 전환 조건 |
|---|---|---|
| 자녀(C2형)이 "엄마, 그거 함량 너무 적어" 개입 | 자녀가 공유한 비교 카드 수신(카카오톡) | 자녀의 설득 + 단순한 비교 결론 |
| 기존 제품 가격 대폭 인상 체감 | "더 싼 거 없나" 처음으로 탐색 시도 | 가격 절감이 명확하게 제시될 것 |
| TV·뉴스에서 특정 성분 유해성 보도 | 검색 → 플랫폼 유입 가능 | 불안이 탐색 동기보다 클 것 |

#### 💬 Emotion — 현재 상태

```
현재 ──────────────────────────────────────────► 계기 발생 시
"만족해" [관성·안정]  "건드리지마" [방어·거부]  "이상하네" [약한 불안]  "자녀에게 물어보자" [의존]
```

#### 🛠 대체 솔루션 (현재)
- TV·라디오 홈쇼핑 정기 구매 (스토리텔링 신뢰)
- 대형마트 진열대에서 익숙한 브랜드 선택
- 지인 추천 그대로 구매

#### ✅ 플랫폼 활용 방향 (직접 전환 목표하지 않음)

| 전략 방향 | 구체 방법 |
|---|---|
| **직접 전환 포기** | N1을 1순위 마케팅 대상으로 삼지 않는다 |
| **간접 바이럴 경로 설계** | 핵심2·A2가 N1에게 **카카오톡으로 공유하는 비교 카드**가 간접 접점 |
| **자녀 대리 탐색 경로 허용** | "어머니 드릴 영양제 찾기" 목적 사용 UX 지원 |
| **전환 트리거 모니터링** | 가격 인상·부정 뉴스 발생 시 비활성1 → Q4 이동 여부 트래킹 |

> 💡 **시장 인식 함의:** N1은 한국 건기식 시장의 35~40%(525~800만 명)를 차지하는 최대 세그먼트. 직접 전환이 아닌 **레몬 마켓 인식 확산**의 간접 효과로 접근한다.
