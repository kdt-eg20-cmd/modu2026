### 핵심1. 한정훈 (36) — "엑셀 비교왕"
**세그먼트:** Q1-A 가성비 최적화자 · Primary Target

| 항목 | 내용 |
|---|---|
| **유형명** | 가성비 파워 비교러 |
| **역할** | IT기업 백엔드 개발자 |
| **주요 문제** | iHerb(달러)·쿠팡(원)·네이버를 탭 8개 열어놓고, 환율 적용한 함량당 단가를 스프레드시트에 직접 기록해 비교한다. 한 제품의 용량별(60정/120정/240정) 단가를 일일이 나눗셈하는 데만 20분을 쓴다. 할인·쿠폰까지 반영하면 비교가 기하급수적으로 복잡해진다. |
| **목표** | "같은 비타민D 1,000IU면, 지금 이 순간 어느 채널이 가장 싼지 5초 안에 알고 싶다." |
| **사용 맥락** | 2~3개월 주기 정기 재구매 시, 또는 iHerb 세일 기간에 채널 간 최저가를 비교할 때 |
| **감정** | 비교 자체는 즐기지만, 반복적인 수동 계산에 **피로감**을 느낀다. "이걸 자동화해주는 도구가 왜 없지?"라는 불만이 지배적. |
| **대체 솔루션** | 개인 스프레드시트, 에누리 건강플러스(부분적), iHerb 내 정렬 기능 |

### 핵심2. 박소연 (43) — "검진 후 첫 구매자"

|---|---|

| **역할** | 중견기업 인사팀 과장 |
| **주요 문제** | 올해 건강검진에서 비타민D 부족 판정을 받았다. 검색하면 "콜레칼시페롤 25μg"이 나오는데 이게 좋은 건지 나쁜 건지 모른다. 가격은 5,000원~50,000원까지 10배 차이가 나고, "비싼 게 좋다"와 "비싼 건 마케팅비"라는 상반된 정보가 쏟아져 불안만 커진다. |

| **사용 맥락** | 건강검진 결과를 받은 직후 1~2주 내, "비타민D 추천" 검색으로 유입 |

| **대체 솔루션** | 네이버 블로그·카페 후기, 약사 유튜버 추천, 약국 방문 상담, 쿠팡 베스트셀러 순 정렬 |

| **대체 솔루션** | Examine.com, 개인 노션 DB, 건강 관련 논문·유튜브, 필라이즈 앱(부분적) |

| 항목 | 내용 |
|---|---|
| **유형명** | 복합 건강 이슈 신중 탐색자 |
| **역할** | 물류회사 팀장 |
| **주요 문제** | 고혈압으로 약을 복용 중이며, 최근 피로감이 심해 영양제를 시작하고 싶다. 그런데 혈압약과 건기식의 상호작용이 걱정된다. 검색하면 "오메가3는 혈압약과 같이 먹으면 안 된다"와 "괜찮다"가 혼재되어 판단이 불가능하다. |

| **사용 맥락** | 병원 처방전을 받은 뒤, 또는 계절 변환기 피로감 증가 시 |
| **감정** | 건강에 대한 **절실함**과 잘못된 선택에 대한 **두려움**이 공존. "차라리 아무것도 안 먹는 게 안전한가" 하는 **회피 충동**. |
| **대체 솔루션** | 약국 약사 상담(대면), 병원 주치의에게 질문, 필라이즈 상호작용 체크 기능 |

| **사용 맥락** | 새 제품을 시도하기 전 검증 정보를 수집할 때. 빈도는 낮지만(분기 1~2회), 검증 통과한 제품에 장기 충성. |
| **감정** | 데이터에 대한 **회의적 냉정함**. 플랫폼이 "독립적"이라고 주장해도 쉽게 신뢰하지 않는다. 하지만 검증 데이터의 근거가 투명하면 **강한 지지자**로 전환된다. |
| **대체 솔루션** | ConsumerLab 유료 구독($59/년), 제조사에 직접 COA 요청, Labdoor 등급 참조 |

| **대체 솔루션** | 인스타·유튜브 인플루언서 콘텐츠, 네이버 검색, 올리브영 매장 직원 상담 |

| **대체 솔루션** | iHerb 정기배송 할인, 쿠팡 로켓배송 재구매, 특별한 비교 도구 없이 기존 패턴 유지 |

### 극단2. 김도현 (29) — "데이터가 틀려서 손해 본 사람"
**세그먼트:** Q1-A 극단 변형 — 플랫폼 신뢰 실패 경험자

| 항목 | 내용 |
|---|---|
| **유형명** | 오류 경험 이탈자 |
| **역할** | 데이터 분석가 |
| **주요 문제** | 기존 건기식 비교 앱에서 "함량 당 최저가"로 추천받아 제품을 구매했는데, 실제로는 1회 섭취량 기준이 잘못 입력되어 있어 함량 대비 비싼 제품을 산 것이었다. 이후 모든 비교 플랫폼의 데이터를 불신하며, 결국 직접 라벨을 확인하고 수동으로 계산하는 방식으로 돌아갔다. |
| **목표** | "데이터의 출처와 정확도를 내가 직접 검증할 수 있는 투명한 시스템이 아니면 다시는 믿지 않겠다." |
| **사용 맥락** | 비교 플랫폼을 사용하더라도 반드시 원본 라벨과 대조하며, 오류 발견 시 즉시 이탈 |
| **감정** | "시간 아끼려다 돈을 버렸다"는 **배신감**. 자동화 도구에 대한 **근본적 회의**. "차라리 내가 직접 하는 게 정확하다"는 **통제 욕구**. |
| **대체 솔루션** | 제품 실물 라벨 직접 확인 + 개인 스프레드시트 |
| **설계 시사점** | 데이터 정확성 해자(moat)의 중요도 검증 — 오류율 5% 이상이면 파워 유저부터 이탈한다는 가설의 현실적 근거. 오류 보고 체계, 데이터 출처 투명 공개, 사용자 교차 검증 기능의 우선순위. |

## 4. 비활성 사용자 (Non-user) — 2명

### 비활성1. 조미라 (58) — "종근당이면 됐지"
**세그먼트:** Q3 수동적 수용자 — 브랜드 의존형

| 항목 | 내용 |
|---|---|
| **유형명** | 브랜드 맹신 무탐색자 |
| **역할** | 아파트 관리사무소 경리 |
| **주요 문제** | 문제가 아니라 **문제를 인식하지 못하는 것**이 문제다. 20년째 종근당 멀티비타민을 홈쇼핑에서 사먹고 있다. 동일 성분·함량의 제품이 3분의 1 가격에 있다는 사실을 모른다. "비싼 게 좋은 거"라고 믿으며, 성분표를 볼 이유를 느끼지 못한다. |
| **목표** | (목표 자체가 부재) "지금 먹는 거 만족하고 있으니 건드리지 마세요." |
| **사용 맥락** | 플랫폼을 사용할 의사 없음. 자발적 유입 가능성 거의 0. |
| **감정** | 현 선택에 대한 **강한 만족과 관성**. 비교 정보가 제시되면 오히려 "내가 바보라는 거냐"라는 **방어적 반응**. 정보 자체를 위협으로 인식. |
| **대체 솔루션** | 홈쇼핑 정기 구매, 지인 추천, 대형마트 진열대에서 익숙한 브랜드 선택 |
| **비활성 이유** | ① 정보 니즈 자체가 부재 ② 성분 리터러시 부족으로 플랫폼 가치를 체감할 수 없음 ③ 현재 소비 패턴에 대한 강한 관성 |
| **활성화 가능 트리거** | 가격 인상 체감, 가족의 건강 이슈("엄마, 그거 함량이 너무 적어"), 건강 뉴스(특정 성분 유해성 보도) → Q3 → Q4 이동 경로 |

### N2. 안형수 (34) — "영양제 자체를 안 믿는 사람"
**세그먼트:** Q3 변형 — 건기식 회의론자

| 항목 | 내용 |
|---|---|
| **유형명** | 건기식 회의론자 |
| **역할** | 스타트업 PM |
| **주요 문제** | "영양제는 비싼 소변"이라고 생각한다. 건기식 시장 자체의 과장 광고, 불투명한 효능 근거, 규제의 허술함을 인지하고 있어 건기식을 소비하지 않는다. 비교 플랫폼이 있어도 "비교할 대상 자체가 의미 없다"는 입장. |
| **목표** | (거부적 목표) "건기식을 사지 않는 것 자체가 합리적 선택이라고 확신한다." |
| **사용 맥락** | 플랫폼 사용 의사 없음. 다만 가족이나 동료가 건기식을 구매할 때 "그거 돈 낭비야"라고 조언하는 역할. |
| **감정** | 건기식 산업에 대한 **냉소**와 **지적 우월감**. "비교해봤자 다 거기서 거기"라는 **무용론**. |
| **대체 솔루션** | 식단 관리, 운동, "음식에서 영양소를 챙기면 된다"는 원칙 |
| **비활성 이유** | ① 건기식 카테고리 자체에 대한 불신 ② "비교"의 전제가 되는 "구매 의도"가 없음 ③ 정보 플랫폼이 산업의 일부라는 인식 |
| **활성화 가능 트리거** | 본인의 건강 검진 이상 소견(Q4-A 진입 계기), 임신·출산 등 생애 전환점, 가까운 사람의 건강 위기 |
